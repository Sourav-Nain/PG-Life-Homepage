# PG Life Homepage

A responsive **PG Life** web application built using **HTML, CSS, Tailwind CSS, PHP, MySQL, and XAMPP**. The project includes a responsive homepage along with a functional **Signup page** that stores user data in a MySQL database.

## Features

- Responsive homepage design
- Signup page with form validation
- User registration using PHP and MySQL
- Tailwind CSS-based responsive Signup UI
- Hero section with background image and gradient overlay
- Responsive search bar
- Major Cities section with hover effects
- Responsive footer
- Font Awesome icons
- Clean and organized code

## Technologies Used

- HTML5
- CSS3
- Tailwind CSS
- PHP
- MySQL
- XAMPP
- Flexbox
- Media Queries
- Font Awesome

## Database

- MySQL database created using XAMPP
- User registration data stored in the database
- PHP used for database connectivity and form handling

- Connecting PHP with MySQL
- Working with XAMPP for local development
- Handling user registration and form submission
- Using Flexbox and Media Queries
- Organizing frontend and backend code efficiently
