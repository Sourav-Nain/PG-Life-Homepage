# PG Life Homepage

A responsive frontend clone of the **PG Life Homepage** built using **HTML** and **CSS**. This project focuses on creating a clean and responsive user interface using modern CSS techniques.

## Features

* Responsive design using Media Queries
* Flexbox-based layout
* Hero section with background image and gradient overlay
* Responsive search bar
* Major Cities section with hover effects
* Responsive footer
* Font Awesome icons
* Clean and organized code

## Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries
* Font Awesome


## Learning Outcomes

* Building responsive web pages
* Using Flexbox for layouts
* Applying Media Queries for different screen sizes
* Creating hover effects and smooth transitions
* Working with background images and gradients
* Organizing HTML and CSS efficiently

## Future Improvements

* Add JavaScript functionality
* Create Login and Signup pages
* Add a responsive navigation menu
* Connect with a backend for dynamic data
